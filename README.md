# 🏗️ BuildTrack - Sistema de Gestión y Control Financiero

**Universidad:** Universidad de Ingeniería y Tecnología (UTEC)  
**Curso:** Introducción a Sistemas de la Información  
**Proyecto:** Prototipo Funcional de Control de Sobrecostos  

## 📖 Descripción del Proyecto
**BuildTrack** es una solución tecnológica diseñada para abordar la falta de visibilidad financiera en tiempo real dentro del sector construcción. El problema central identificado es un **sobrecosto operativo del 18%** derivado del uso de sistemas manuales fragmentados y archivos Excel aislados que no permiten una comunicación fluida entre la obra y la gerencia.

Este sistema centraliza la información, permitiendo que cada gasto registrado en el campo impacte inmediatamente en los indicadores financieros globales de la empresa.

## 🚀 Funcionalidades del Prototipo
El prototipo es una aplicación web interactiva vinculada a una base de datos relacional que permite:

1.  **Gestión de Roles:** Diferenciación de accesos entre el personal de campo (Residentes) y la oficina central (Gerencia).
2.  **Registro de Gastos en Tiempo Real:** Interfaz móvil-friendly para capturar costos de materiales, logística e imprevistos en el momento en que ocurren.
3.  **Dashboard de Control (KPIs):** Visualización automática del presupuesto base vs. gasto real acumulado, con alertas visuales (semáforo rojo) cuando se detectan desviaciones y sobrecostos.
4.  **Trazabilidad:** Historial completo de transacciones con detalles de proveedores y conceptos de gasto.
